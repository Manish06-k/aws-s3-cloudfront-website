# aws-s3-cloudfront-website


## Project Overview

This project demonstrates how to host a static website on Amazon S3 and deliver it securely using Amazon CloudFront.

## Technologies Used

* Amazon S3
* Amazon CloudFront
* HTML5
* CSS3

## Features

* Static website hosting
* Fast content delivery using CloudFront
* Responsive website design
* Simple HTML and CSS implementation

## Project Structure

```text
.
├── index.html
├── style.css
└── images/
```

## Architecture

```text
User
   │
   ▼
CloudFront
   │
   ▼
Amazon S3
```

## Deployment Steps

1. Create an Amazon S3 bucket.
2. Upload the website files.
3. Enable Static Website Hosting.
4. Configure the bucket policy for public access.
5. Create a CloudFront distribution.
6. Access the website using the CloudFront URL.

* Learned how to host a static website on Amazon S3.
* Configured CloudFront to improve performance.
* Understood bucket policies and static website hosting.
* Gained hands-on experience with AWS services.
